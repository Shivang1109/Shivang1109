## ⭐ Flagship Project

### 🛡️ Agentic Scam Intelligence System
> *Autonomous AI honeypot that engages scammers, extracts fraud intelligence & generates threat reports*

An agentic backend system that simulates vulnerable personas to bait scammers into conversation, then automatically extracts structured fraud intelligence — phone numbers, payment IDs, fake orgs, scam patterns — and generates JSON threat reports for cybercrime teams.

| Feature | Details |
|---|---|
| 🤖 Autonomous agents | Persona-based conversational state machine |
| 🔍 Entity extraction | Phone numbers, payment IDs, URLs, organizations |
| ⚠️ Risk scoring | Multi-factor explainable scoring (0–100) |
| 📊 Threat reports | Structured JSON output for cybercrime teams |
| 🚀 Deployed | Live API on Render with Docker + PostgreSQL |

**Tech Stack:** `TypeScript` `Express.js` `PostgreSQL` `Docker` `NLP` `AI Agents`  
🔗 [View Repo](https://github.com/Shivang1109/scam-intelligence-system) • 🌐 [Live Demo](https://scam-intelligence-system.onrender.com/)
